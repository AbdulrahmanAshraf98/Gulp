### Gulp Basics
        Top Level Function
            -[1]-task
            -[2]-src
            -[3]-dest
            -[4]-watch
        
        -[1]-task used to create a new  task
        -[2]-src  used to get file and modify 
        -[3]-dest used to put output 
        -[4]-watch used to watch files and modify

        Function src,dest can take Array as argument
